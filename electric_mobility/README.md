# Modelamiento de Demanda Eléctrica para Flotas de Vehículos en Galápagos

## Estructura del proyecto

```
electric_mobility/
├── Anexos/          # Documentación de respaldo
└── Code/            # Código de simulación
```

---

## Anexos

La carpeta `Anexos/` contiene toda la documentación de soporte del proyecto:

### Resumen del proyecto
- **`RESUMEN.docx`** — Documento con el resumen completo del proyecto: objetivos, metodología, resultados y conclusiones.

### Justificación del análisis por isla
- **`justificacion perfil de actividad.docx`** — Justificación del perfil de actividad utilizado para cada isla (Santa Cruz, San Cristóbal e Isabela), incluyendo los criterios de selección de ciclos de manejo y los supuestos adoptados.

### Datasheets de vehículos seleccionados
| Vehículo | Archivo |

| Bicicleta Eléctrica EKKO 4 | `BICICLETA ELÉCTRICA EKKO 4 - Ecomove.pdf` |
| Moto Eléctrica XZ6 | `MOTO ELÉCTRICA XZ6 - Ecomove.pdf` |
| Kia EV5 | `kia-ev5-brochure.pdf` |
| Riddara RD6 2025 | `Riddara-RD6-2025-LK.pdf` |

## Código

La carpeta `Code/` contiene todos los scripts de simulación en Python. Consulta el archivo **`Code/README.md`** para ver la descripción detallada de cada módulo, instrucciones de ejecución y dependencias.
